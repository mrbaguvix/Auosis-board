## AUOSIS-board

Shows data overview from mongodb and paginates it on Table or Grid.

## Usage

Use your favorite package manager to install project dependencies:

```bash
$ yarn
$ npm install
```

Then run the project with:

```bash
$ yarn start
$ npm start
```

## Tooling with
- ReactJS + Hooks
- Styled components
- Apollo Client (Graphql)
- Typescript
- UiKit
